# Healthcare Analytics Dashboard

## Project Overview
This project analyzes healthcare data to uncover trends in patient visits, encounter types, procedures, and insurance cost distribution.  
The dashboard helps understand healthcare utilization patterns and financial impact.

## Tools Used
- SQL Server (Data extraction & analysis)
- Power BI (Dashboard & visualization)
- Excel/CSV datasets

## Key Insights
- Total patient visits trend across years
- Distribution of encounter types (inpatient, emergency, outpatient, etc.)
- Insurance-wise cost contribution
- Total procedures and patient counts

## Files Included
- Power BI Dashboard (.pbix)
- SQL analysis queries
- Healthcare datasets (CSV)
- Dashboard screenshot

## Objective
To demonstrate skills in SQL querying, data modeling, and Power BI dashboard creation using a real-world healthcare dataset.
